Politique de Confidentialité

Dernière mise à jour : 1er décembre 2024

Cette politique de confidentialité décrit comment PPE ECE ("nous", "notre" ou "l'application") collecte, utilise et protège vos informations personnelles.

1. Informations collectées

Nous collectons les informations suivantes :

  • Informations de compte : email, nom, prénom, mot de passe (chiffré)
  • Profil étudiant : année d'études, matières favorites, objectifs d'apprentissage
  • Contenu utilisateur : cours uploadés, QCMs générés, flashcards créées
  • Données d'utilisation : résultats de QCM, progression d'apprentissage

2. Utilisation des données

Vos données sont utilisées pour :

  • Créer et gérer votre compte utilisateur
  • Stocker vos cours et générer du contenu pédagogique (QCMs, flashcards)
  • Suivre votre progression d'apprentissage
  • Améliorer l'application et ses fonctionnalités

3. Partage des données

Nous ne vendons pas vos données personnelles. Vos données peuvent être partagées avec :

  • Supabase : hébergement de la base de données (serveurs sécurisés)
  • OpenAI/Anthropic : traitement IA pour la génération de QCMs et flashcards (le contenu de vos cours peut être envoyé pour analyse)

4. Sécurité

Nous prenons la sécurité de vos données au sérieux :

  • Mots de passe chiffrés avec bcrypt
  • Communications sécurisées via HTTPS
  • Authentification par token JWT
  • Base de données hébergée sur infrastructure sécurisée

5. Conservation des données

Vos données sont conservées tant que votre compte est actif. Vous pouvez supprimer votre compte et toutes vos données à tout moment depuis les paramètres de l'application.

6. Vos droits

Conformément au RGPD, vous disposez des droits suivants :

  • Accès : consulter vos données personnelles
  • Rectification : corriger vos informations
  • Suppression : supprimer votre compte et toutes vos données
  • Portabilité : exporter vos données

7. Modifications

Nous pouvons mettre à jour cette politique de confidentialité. Les modifications seront publiées sur cette page avec une nouvelle date de mise à jour.

8. Contact

Pour toute question concernant cette politique de confidentialité ou vos données personnelles, contactez-nous à :
